## Polkadot SDK parachain template

This is a fork of the [Polkadot SDK parachain template](https://github.com/paritytech/polkadot-sdk-parachain-template/), a template to get you up-and-running on Polkadot.

This template is also available in the list of templates from [Pop CLI](https://onpop.io/cli/).

The SDK version is stable2503-3.
It's using some of the latest features like:
- umbrella crate (polkadot-sdk)
- construct runtime v2 (#\[runtime::runtime\])
